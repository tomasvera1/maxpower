Contactores A-Line

ABB posee una serie de Contactores y Relés adecuados para una amplia variedad de aplicaciones en edificios, industrias., para el control de motores, calefacción, aerogeneradores, aire condicionado, bombas, equipos elevadores, iluminación, corrección del factor de potencia de procesos, etc… La gama de contactores A-line tiene el mismo diseño desde 9A a 750A (servicio AC3) cargas de motor – y para una amplia gama de aplicaciones CC y otras aplicaciones CA. Esto le hace más fácil satisfacer sus necesidades con menos componentes. La gama incluye minicontactores de 4 y 5,5kW, contactores compactos hasta 400kW, contactores de instalación para uso doméstico e industrial, contactores de barrote, relés térmicos y electrónicos, contactores server y una amplia gama de accesorios que aseguran flexibilidad y adaptabilidad a las especificaciones del cliente.

Este es el producto de las últimas ideas sobre diseño eléctrico e instalación de componentes, incorporando los desarrollos más recientes en aspectos de seguridad, materiales y electrónica. Esta nueva gama ofrece un concepto total de sistema. La A-Line, junto con las protecciones contra cortocircuitos con o sin fusibles de ABB y relés de sobrecarga electrónicos proporcionan la base para una nueva generación de aparatos diseñados para una comunidad de sistemas ABB-Line.

Nuestro objetivo es proporcionar a los diseñadores, ingenieros, instaladores y usuarios finales las mejores soluciones para sistema posibles.

  • ABB- Line – un concepto de sistema.
  • Diseñado para una instalación y un mantenimiento sencillos y seguros.
  • Fácil de integrar con otros componentes del sistema.
  • Diseño compacto.
  • Una gama completa de accesorios.
  • Cumple con las últimas normas internacionales y nacionales.
  • Rendimiento ambiental mejorado.
